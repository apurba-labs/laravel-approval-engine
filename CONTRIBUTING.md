# Contributing to Approval Engine

Thank you for considering contributing! 

## Development Setup
1. Fork the repository.
2. Install dependencies: `composer install`.
3. Setup your test database (MySQL, PostgreSQL).
4. Run tests to ensure everything is green: `./vendor/bin/phpunit`.

## Pull Request Process
1. Create a feature branch: `git checkout -b feature/amazing-feature`.
2. Ensure all tests pass.
3. Add tests for any new logic.
4. Submit your PR against the `dev` branch.

## Coding Standards
Please follow [PSR-12](https://www.php-fig.org) coding standards and provide clear commit messages using [Conventional Commits](https://www.conventionalcommits.org).
